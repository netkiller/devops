DevOps Tools
====

OS Software Configure Managment

Install
-------
	$ git clone https://github.com/oscm/devops.git
	$ cd devops
	$ sudo python3 setup.py sdist
	$ sudo python3 setup.py install

	or 
	
	$ sudo python setup.py install --prefix=/srv/devops
